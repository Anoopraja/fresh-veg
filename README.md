🥦 Fresh Veg UI (Frontend Practice Project)

A simple frontend project built while learning HTML, CSS, and JavaScript.
This project focuses on layout design, responsiveness, and basic UI interactions.

🔗 Live Demo

👉 https://fresh-veg-sigma.vercel.app/

⚙️ Tech Stack
HTML5
CSS3
JavaScript 

📌 Features
Responsive layout (mobile + desktop)
Clean UI design for a vegetable store
Basic interactivity using JavaScript
Flexbox-based layout structure
🎯 Purpose of This Project

This is not a full e-commerce website.
The goal was to:

Practice frontend development
Understand layout systems (Flexbox)
Improve UI building skills
Learn basic DOM manipulation
🚀 What I Learned
How to structure a webpage properly
Handling responsiveness issues (like unwanted horizontal scroll)
Using Flexbox for layout
Basic event handling in JavaScript
⚠️ Limitations
No backend
No database
No authentication system
Not production-ready
🛠️ Future Improvements
Add backend (Node.js / Firebase)
Implement authentication
Connect real product data
Improve UI/UX

If you find bugs or have suggestions, feel free to open an issue or connect.
